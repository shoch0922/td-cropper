# td-cropper
画面の特定領域をクロップするためのtoxです。

## 使い方
<img src="https://user-images.githubusercontent.com/10994053/153734904-3519a110-2d51-42ab-9fd2-ca5a3c97b56b.png" width="200px">

第1入力のクロップしたいTOPをつなぐ。<br>

### パラメータ
- center: 画面の中央をトリミングしたい場合ONにする
- width: クロップ領域の横幅
- height: クロップ領域の高さ
- x: クロップ領域の左端からの座標
- y: クロップ領域の上端からの座標
